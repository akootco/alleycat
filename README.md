# AlleyCat
This plugin aims to handle core administrative applications.
## Features
- Bans are recorded in a database instead of a file.
## Commands
- `/kick`
- `/ban`
- `/tempban`
- `/queryban`
- `/banlist`
- `/uuid`
## API
- `kickPlayer(...)`
- `banPlayer(...)`
- `pardonPlayer(...)`
- `getBanQuery(...)`
